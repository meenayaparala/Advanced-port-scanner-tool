# 🔍 Advanced Port Scanner

A GUI-based port scanner application built with Python. This tool allows users to scan open ports on a target host and save detailed scan reports. It provides an intuitive interface for ethical hackers, cybersecurity learners, and IT professionals to test and analyze network exposure.

---

## 📌 Features

- ✅ **Graphical User Interface** (built with Tkinter)
- 🔎 **Custom Port Range Scanning**
- 📡 **Real-time Status Updates**
- 📝 **Exportable Scan Reports** (TXT/CSV)
- 🚫 **Handles Invalid or Unreachable Hosts**
- 💡 **Lightweight and Beginner-Friendly**

---

## 🛠 Technologies Used

- **Python**
- **Tkinter** (GUI)
- **Socket Programming**
- **CSV/Text File Handling**

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/yourusername/advanced-port-scanner.git
cd advanced-port-scanner
````

### 2. Install dependencies

No external libraries are required other than Python standard modules.

### 3. Run the application

```bash
python port_scanner_gui.py
```

> Make sure you have Python 3.x installed on your machine.
